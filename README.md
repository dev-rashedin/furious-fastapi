# Furious FastAPI


This repository documents my complete journey learning **FastAPI** — from basic concepts to real-world applications.

The goal is not just to learn the framework, but to understand how to build **production-ready Python backends**, experiment with ideas, and explore modern backend patterns including **AI-powered APIs**.

---

## Why this repository exists

I come from a **Node.js backend background**, and this repo is my structured path to mastering FastAPI and the Python ecosystem.

Instead of scattered notes and experiments, everything lives in one place:

- Learning notes
- Official documentation exploration
- Small experiments
- Real backend projects
- AI-powered API projects

By the end of this journey, I aim to be fully comfortable building scalable APIs with FastAPI.

## Learning Roadmap

The journey roughly follows this order:

1. Python fundamentals needed for backend development
2. FastAPI core concepts
3. Request validation with Pydantic
4. Dependency injection
5. Database integration
6. Authentication and security
7. Async programming
8. Production deployment
9. AI / LLM API integrations
